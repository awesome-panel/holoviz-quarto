Introduction

The purpose of this project is to make it easy to use the HoloViz ecosystem with Quarto.

We do this by providing the quarto-holoviz extension as well as the how-to guides listed in the sidebar.

With the HoloViz ecosystem you can easily embed the PyData ecosystem live and interactively in your documentation.

Interactive Matplotlib

Code 
import numpy as np
import panel as pn
import matplotlib.pyplot as plt

pn.extension(design="material")

def plot(factor):
  r = np.arange(0, 2, 0.01)
  theta = factor * np.pi * r
  fig, ax = plt.subplots(
    subplot_kw = {'projection': 'polar'} 
  )
  ax.plot(theta, r)
  ax.set_rticks([0.5, 1.2, 1.5, 2])
  ax.grid(True)
  plt.close(fig)
  return fig

slider = pn.widgets.FloatSlider(value=2, start=1, end=4, step=0.25, name="Factor")
plot = pn.pane.Matplotlib(pn.rx(plot)(slider), tight=True, format="svg", height=400, sizing_mode="stretch_width")
pn.Column(
  slider, plot, sizing_mode="stretch_width"
).embed(max_states=16, max_opts=16)

# Quarto-HoloViz

## Introduction

The purpose of this project is to **make it easy to use the [HoloViz](https://holoviz.org/) ecosystem with [Quarto](https://quarto.org/)**.

We do this by providing the `quarto-holoviz` extension as well as the *how-to* guides listed in the sidebar.

With the HoloViz ecosystem you can easily **embed the PyData ecosystem live and interactively in
your documentation**.

## Examples

### Interactive Matplotlib

```{python}
#| code-fold: true
#| code-summary: "Code"

import numpy as np
import panel as pn
import matplotlib.pyplot as plt

pn.extension(design="material")

def plot(factor):
  r = np.arange(0, 2, 0.01)
  theta = factor * np.pi * r
  fig, ax = plt.subplots(
    subplot_kw = {'projection': 'polar'} 
  )
  ax.plot(theta, r)
  ax.set_rticks([0.5, 1.2, 1.5, 2])
  ax.grid(True)
  plt.close(fig)
  return fig

slider = pn.widgets.FloatSlider(value=2, start=1, end=4, step=0.25, name="Factor")
plot = pn.pane.Matplotlib(pn.rx(plot)(slider), tight=True, format="svg", height=400, sizing_mode="stretch_width")
pn.Column(
  slider, plot, sizing_mode="stretch_width"
).embed(max_states=16, max_opts=16)
```
